Migrate from 1.4.8 to 2.0.0

This is a quick guide of the changes made for BooruSharp 2.0.0
For more information on BooruSharp 2.0.0 see the documentation page.

Function names are now marked as Async

This means most functions now end with "Async":
GetRandomImage -> GetRandomImageAsync
GetTags -> GetTagsAsync
etc...

CustomBooru was removed for inheritance

There are now "Template" located in the namespace BooruSharp.Booru.Template.
You just have to inherite from the class you need.

Sankaku Complex was added

You can now do searchs on Sankaku Complex.

Some methods were removed/added

  • GetImageByMd5Async was added
  • GetNbImage was removed
  • GetLimit was removed
  • GetImage was removed
Some others internal methods were hidden from the inspector
