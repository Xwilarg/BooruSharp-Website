Introduction
Welcome on BooruSharp's documentation.

BooruSharp is a C# library allowing you to get images from Booru websites. BooruSharp currently handle the following websites:
  • booru.allthefallen.moe
  • danbooru.donmai.us
  • e621.net
  • e926.net
  • furry.booru.org
  • gelbooru.com
  • konachan.com
  • lolibooru.moe
  • realbooru.com
  • rule34.xxx
  • safebooru.org
  • sakugabooru.com
  • beta.sankakucomplex.com
  • xbooru.com
  • yande.re
